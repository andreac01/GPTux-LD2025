Contenuto del corso

In questo corso vedremo velocemente la teoria e la pratica dietro ad un modello Transformer. Il corso prevede due parti principali:

  • Introduzione ai Transformer e GPT: Cos'è un Transformer, attenzione, tokenizzazione, embedding, layer norm, MLP.
  • Notebook interattivo: Implementazione dei componenti di un Transformer, mini-training e visualizzazioni.

NOTA: Per seguire il corso non è necessario scaricare ed utilizzare il notebook , tuttavia è consigliato se si vuole sperimentare in prima persona.

Setup rapido su Linux (Facoltativo)

  1. Clona la repository:
    git clone https://github.com/andreac01/GPTux-LD2025.git
  2. Installa Python (consigliato 3.11+):
    sudo apt update
sudo apt install python3 python3-venv python3-pip
  3. Crea ambiente virtuale:
    cd ~
python3 -m venv .venv/gptux-env
source ~/.venv/gptux-env/bin/activate
  4. Installa pacchetti:
    pip install --upgrade pip
pip install -r requirements.txt
pip install jupyter
  5. Avvia Jupyter Notebook:
    jupyter notebook
